# latex-thesis-template
Una plantilla de tésis en LaTeX enfocada en ingeniería y desarrollo de hardware y software.

```Estructura de archivos completa para proyecto de tesis
proyecto-tesis/
  ├── main.tex                   # Documento principal de la tesis
  ├── config/
  │   ├── variables.tex          # Variables globales (título, autor, etc.)
  │   ├── preamble.tex           # Configuración y paquetes
  │   └── macros.tex             # Comandos personalizados
  ├── chapters/                  # Capítulos de la tesis
  │   ├── 01-planteamiento.tex
  │   ├── 02-marco-teorico.tex
  │   ├── 03-metodologia.tex
  │   ├── 04-desarrollo.tex
  │   ├── 05-resultados-discusion.tex
  │   └── 06-conclusiones-recomendaciones.tex
  ├── assets/                    # Recursos estáticos
  │   ├── images/                # Imágenes y fotografías
  │   ├── diagrams/              # Archivos fuente de diagramas
  │   ├── code/                  # Código fuente para incluir
  │   ├── data/                  # Conjuntos de datos para análisis
  │   └── hardware/              # Diseños de hardware/circuitos
  └── references/                # Referencias bibliográficas
      ├── bibliography.bib       # Base de datos de referencias
      ├── Acronyms.bib           # Acrónimos
      └── glossary.tex           # Glosario de términos técnicos
```
